인공지능 연구분야

요소 기술 분야 - 탐색
- 문제의 답이 될 수 있는 것들의 집합을 공간으로 간주하고 문제에 대한
최적의 해를 찾기 위해 공간을 체계적으로 뒤지는 것
- 대부분의 탐색 방법은 탐색 공간에 대한 정보를 그래프 자료구조로 만들어 탐색을 진행

요소 기술 분야 - 지식표현
- 문제를 해결하거나 심층적인 추론을 하는 데 사용할 수 있도록
지식을 효과적으로 표현하는 방법
IF-THEN 형태의 규칙(rule), 슬롯(slot)과 필러(filler)로 구성된 프레임(frame)

요소 기술 분야 - 추론
- 가정이나 전제로부터 결론을 이끌어 내는 것
- 지식이 규칙으로 표현되고 규칙이 조건부를 만족하는 입력이 주어지면
결론부의 내용이 추론 결과로 얻어짐
- 추론방식에는 "전향추론"과 "후향추론"이 존재

★요소 기술 분야 - 학습
- 경험을 통해 동일한 문제 또는 유사한 문제를 더 잘 해결할 수 있도록
시스템의 구조나 파라미터를 바꾸는 것
- 현재 인공지능 모델 개발에 가장 많이 사용하는 "머신러닝"및 "딥러닝"에 사용되는 방법
- 지도학습, 비지도학습, 강화학습
  지도학습 : 문제와 정답을 모두 알려주고 공부를 시키는 방법(예측, 분류)
  비지도학습 : 답을 가르쳐주지않고 공부시키는 방법(연관 규칙, 군집)
  강화학습 : 보상을 통해 상은 최대화, 벌은 최소화하는 방향으로
	       행위를 강화하는 학습 (보상)

요소 기술 분야 - 계획수립
- 현재 상태에서 목표 상태에 도달하기 위해 수행해야할 일련의 행동 순서를 찾는 것
- 계획수립 또한 많은 계획 중 최선의 계획을 찾는 문제기 때문에
   일종의 "탐색"문제이기도 함
- 계획수립의 주요 관심사는 "시간과 같은 비용을 최소로하는 계획 찾기"

전문가시스템
우리가 이미 나열해놓은 지식을 통해 추론하는 

1940년대(McCulloch, Pitts)
인간신경 세포 수학적 모델링
- 0과 1로 이루어진 2진법 논리모델로 뉴런의 작용을 설명
- 인간의 뇌를 기계적으로 모델링한 최초의 논문
- 인공의 뉴런을 AND, OR, NOT 명제로 연결
- 15년 후 퍼셉트론이라는 개념으로 이어짐

1950년(Rosenblatt)
퍼셉트론
- 1957년 "로젠블럿"에 의해 제안
- 신경세포를 계산적 모델로 표현한 "최초의 신경망"
- 맥컬록-피츠 모델을 기반으로 만들어진 가장 간단한 인공 신경망 구조
- 수용층, 연합층, 반응층의 세부분으로 구성
- AND, OR 연산 등 선형 분리가 가능한 문제의 해결에만 사용 됨

1960년대(Minsky)
XOR 문제
- 1969년 로젠블럿의 동창인 마빈 민스키 박사에 의해 제안
- 비선형 문제는 해결할 수 없는 퍼셉트론의 한계를 수학적으로 증명
- 퍼셉트론으로는 XOR문제를 해결 할 수 없음
- 해당 사건으로 인공지능의 1차 겨울이 시작

1970년대
전문가 시스템
- 문제 자체에 대한 지식을 활용하기 보다, "수단-목표 분석"과 같은 기법을 사용하여
해를 기계적으로 탐색하는 연구
- 규모가 작은 문제에는 만족스럽지만, 큰 규모 문제에는 적용 불가
- 따라서, 범용 방법보다는 특정 영역의 문제를 효과적으로 해결하는 방법으로 주로 연구
- 특정 영역의 문제에 대해 전문가 수준의 해답을 제공하는 "전문가 시스템"

1980년대(Hinton)
멀티 퍼셉트론
- 단층 퍼셉트론은 선형분류만 가능하기 때문에 XOR 게이트 연산 불가능
- "멀티 퍼셉트론"은 비선형인 XOR 게이트 연산 가능
- 기존 단층 퍼셉트론의 입력층과 출력층 사이에 은닉층(hidden layer)을 추가
- XOR게이트는 기존의 AND, NAND, OR 게이트를 조합하면 해결 할 수 있기 때문에
층을 계속 추가하여 만들 수 있음

2000년대(Hinton)
깊은 신경망 모델(Deep learning)
- 멀티 퍼셉트론과 역전파(BackPropagation) 알고리즘 활용
- Layer, 활성화 함수, 손실함수, 옵티마이저 등으로 구현
- 다양한 신경망 모델 개발(DNN,CNN, RNN, GAN, Attention 등)
